# 🛒 TechStore - E-Commerce Website

A modern, responsive E-Commerce website built using HTML, CSS, and JavaScript. This project demonstrates a complete frontend shopping experience with dynamic product loading, search, filtering, shopping cart functionality, dark mode, and responsive design.

---

🌐 **Live Website:** https://tech-store-six-cyan.vercel.app/

## ✨ Features

- 🏠 Responsive Home Page
- 🛍️ Products Page
- 🔍 Product Search
- 📂 Category Filter
- 🛒 Shopping Cart
- 💾 Local Storage Support
- 🌙 Dark / Light Mode
- 📱 Responsive Navigation
- 📄 About Page
- 📞 Contact Page
- 📦 Dynamic Product Loading using Fetch API
- 📁 JSON-based Product Data
